# example base — homebase showcase

Fictional workspace of "Alex Stark", an IT generalist: day-job
cloud/k8s, evenings in the homelab, weekends on embedded /
home-automation / scratch experiments.

Used to take screenshots and demo homebase features. Files inside
each project are stubs — only enough to make property detectors,
tags, archive, worktrees, and notes light up.

```sh
BASE_FOLDER=$PWD/example uv run b
BASE_FOLDER=$PWD/example uv run b ls -l
BASE_FOLDER=$PWD/example uv run b ls "##homelab"
BASE_FOLDER=$PWD/example uv run b ls --archived
```
